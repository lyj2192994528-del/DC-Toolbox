# DC Toolbox 功能说明

本文档对应 DC Toolbox v1.0.1。标记为“规划中”的模块目前仅提供独立入口和能力说明，不代表已经接入真实硬件或网络协议栈。

## 串口终端

- 扫描 Windows 串口，并显示端口号、制造商、VID/PID 和序列号。
- 设置波特率、数据位、停止位、校验位和流控，支持自定义波特率。
- 支持 DTR、RTS、Break 控制、设备重新插入后自动重连和异常断开提示。
- 接收支持 ASCII/HEX、UTF-8/GBK/Latin-1、时间戳、搜索、暂停显示、自动滚动与收发统计。
- 默认使用固定 20 ms 空闲超时分帧，也可选择自适应超时或按 LF、CR、CRLF 分帧。
- 连续高速数据最迟每 100 ms 刷新一次，避免没有空闲间隔时界面长期不显示。
- 发送支持 ASCII、HEX、BIN、DEC，支持追加换行、SUM8、XOR8、CRC16-Modbus 校验及定时发送。
- 多条发送支持 20 条起步、最多 100 条；按已选条目逐条顺序循环，而不是合并后一起发送。

## 实时波形

- 支持 FireWater/CSV、NamedData、JustFloat 三种协议。
- 最多 16 个通道，可修改名称、颜色和显示状态。
- 缓存和显示点数可在 1～50000 范围设置，并支持鼠标滚轮缩放与拖动查看。
- 支持通道统计、暂停、清空、截图和 CSV 导出。
- 协议说明直接显示在波形页面；解析错误会单独计数，不阻塞后续数据。

## 数据记录与分析

- 原始串口数据以 BIN 保存，并生成同名 JSON 元数据，保留记录时间和串口参数。
- CSV 分析支持粘贴或导入数据、列识别和基础统计。
- 波形数据可导出为 CSV，便于在 Excel、Python 或其他分析工具中继续处理。

## 数据工具

- 进制转换：BIN、OCT、DEC、HEX 联动转换，支持负数、下划线分组和大整数。
- CRC/校验：提供常用校验算法与标准测试结果显示。

## 开发计算工具

- 欧姆定律：在电压、电流、电阻、功率中任意填写两项，计算其余两项。
- 功率计算：直流、纯阻性负载、单相/三相交流与效率换算，并支持反算。
- 串并联电阻：动态增删、批量选择、等效阻值计算及缺失电阻反算。
- 串并联电容：等效电容量计算及缺失电容反算。
- 电容换算：pF、nF、μF、mF、F 互换，并生成常用三位电容代码。
- LED 限流：计算串联电阻，推荐 E24 标准值和安全功率档位。
- 同相运放：计算闭环增益和输出电压，检查供电范围，并支持目标增益反算。
- 电阻分压：计算或反算 VIN、VOUT、上臂电阻、下臂电阻，并显示静态电流和功耗。

## 虚拟串口

- 检测本机 com0com 安装与管理工具，提供端口对工作原理和安全提示。
- DC Toolbox 不捆绑、不修改、不静默安装第三方内核驱动。
- 建议只使用与当前 Windows 版本兼容且具有有效数字签名的 com0com 版本。

## 界面与国际化

- 首次运行可选择简体中文或 English，主界面右上角可随时独立切换。
- 工具导航、功能页面、关于信息和底部推广栏均随语言同步切换。
- 支持窗口缩放和全屏布局；首次打开使用适中窗口并显示启动动画、图标和版本号。

## 规划中的模块

以下入口已按模块化结构预留，但 v1.0.1 尚未提供完整的真实设备或网络通信能力：

- BLE 扫描、连接、Notify 与 OTA。
- USB/HID/CDC 设备管理与数据收发。
- TCP、UDP、MQTT 网络调试。
